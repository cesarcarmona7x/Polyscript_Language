El proyecto requiere las siguientes librerías para poder ser compilado (Ya están incluidas en el directorio):
-Boost 1.66.0 https://www.boost.org/users/history/version_1_66_0.html
-FTGL https://sourceforge.net/projects/ftgl/ (Se anexó una versión ligeramente modificada para el proyecto).
-STB https://github.com/nothings/stb
-GLEW http://glew.sourceforge.net/

El uso de DirectX 11 hace que requiera mínimo Windows 7 para ejecutar el proyecto.
El ejecutable correcto para ejecutar fuera de Visual Studio es el que está en la segunda carpeta de Polyscript_Language
Se usó Visual Studio 2012 para la compilación del proyecto. Cualquier versión superior debería funcionar, pero no se recomienda actualizar la versión
del proyecto si se usa una versión superior.
